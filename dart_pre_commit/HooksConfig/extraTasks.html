List<TaskBase>? extraTasks
inherited

A list of additional tasks to be added to the hook.

These are added in addition to the four primary tasks. They are always added last to the hook, so they will also run last. If you need more control over the order, instanciate the primary tasks by hand, using HooksProviderInternal
