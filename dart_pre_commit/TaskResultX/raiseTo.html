raiseTo method Null safety

TaskResult raiseTo(
  1. TaskResult target
)

Raises the hook result to a more severe level, if required.

Compares this with target and returns the more severe result. The results severity is as according to the following table, the most severe result at the top:

  • TaskResult.rejected
  • TaskResult.modified
  • TaskResult.accepted

So, if for example, you would call:

TaskResult.modified.raiseTo(TaskResult.rejected)

if would return TaskResult.rejected.

Implementation

TaskResult raiseTo(TaskResult target) => target.index > index ? target : this;
